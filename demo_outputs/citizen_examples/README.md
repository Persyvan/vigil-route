# Citizen Mode Examples

Screenshots from the 311 app simulation showing real-time road damage detection from smartphone photos.

## Features Demonstrated

- **Automatic geolocation**: GPS coordinates captured with each photo
- **AI classification**: Pothole, crack, and surface deformation detection
- **Risk assessment**: Critical, moderate, or low severity levels
- **Action recommendations**: Immediate repair, monitoring, or routine maintenance

## How to View

The images display directly in this folder. Each screenshot shows:
- Input image with detected damage
- AI classification and confidence score
- Risk level indicator
- Recommended action
- GPS coordinates and vehicle speed (if applicable)

These examples demonstrate how citizens can report infrastructure issues quickly using their smartphones.
